CY_2024.11.15

Atmospheric Entry (Inicio del Proyecto)

Se decidió la temática "Cyberpunk Espacial" para representar la unión del equipo. Se estableció la paleta de colores basada en contrastes oscuros (#131314) con acentos neón (Rosa y Cyan) para asegurar buen contraste visual.

CY_2024.11.18

Sonic Calibration (Refactor a CSS)

Dificultad encontrada: Migrar los mockups generados con Tailwind a CSS puro manteniendo el Responsive Design. Solución: Se implementó la metodología BEM y el uso intensivo de variables CSS nativas (`:root`) para controlar los tres breakpoints exigidos por la cátedra sin perder consistencia.

Crew Assembly (Estructuras Nativas)

Integración de HTML5 semántico moderno. Se reemplazaron scripts complejos por el uso de la etiqueta <details> para los acordeones en los perfiles de usuario, mejorando la accesibilidad y el SEO.
